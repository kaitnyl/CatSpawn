Cat Spawn

Call the image generator like:

<img src="http://misc.io/catspawn/?250x200">

If you want to filter the type of image, you would do so like:

<img src="http://misc.io/catspawn/?250x200.png">

By default, it will return .png. Only .png or .jpg are allowed.
Known bug about random broken images- waiting for response.

All images must display the source url (that white text) to avoid copyright issues.
